# Astronomy Articles Site

## Overview
This project is a modern and interactive website dedicated to astronomy articles. Users can create and publish articles that include a title, a main illustration image, text content, and up to three additional images. The site is designed with a dark theme inspired by space, featuring animations and a responsive layout.

## Features
- Create and publish articles with:
  - Title
  - Main illustration image
  - Text content
  - Up to three additional images
- Articles are displayed immediately upon publication without reloading the page.
- Modern design with a dark theme and space-inspired aesthetics.
- Light animations for article appearance and interactions.
- Responsive design for mobile compatibility.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open `index.html` in a web browser to view the site.
3. Use the article creation form to publish new articles.

## Usage
- Fill in the title and content fields.
- Upload or provide URLs for the main image and additional images.
- Click the "Publish" button to display the article on the page.
- Articles will be displayed in reverse chronological order.
